# Quickstart Custom

默认使用 `FishWebSocketAdapter` 的 actor-first 示例，展示如何使用 `ActorPluginBuilder` 把 `MessageContext` 路由到自定义 actor。

运行：

```bash
cargo run -p fish-example-quickstart-custom
```

导入浏览器 Cookie 并启动：

```bash
cargo run -p fish-example-quickstart-custom -- --cookies "cookie2=...; unb=...; _m_h5_tk=..."
```
