# Quickstart Simple

默认使用 `FishWebSocketAdapter` 的宏插件示例，展示最小宿主如何通过 `RuntimeHost` 组装一个基于 `#[plugin]` / `#[message]` 的插件。

运行：

```bash
cargo run -p fish-example-quickstart-simple
```

导入浏览器 Cookie 并启动：

```bash
cargo run -p fish-example-quickstart-simple -- --cookies "cookie2=...; unb=...; _m_h5_tk=..."
```
